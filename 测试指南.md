## 观察操作与结果对应

### 前端 → 后端 → 前端 完整流程

```
用户操作          前端行为            后端处理               前端响应
----------       ----------          ----------            ----------
点击场景按钮  →  填充表单数据     →  (无)               →  表单更新
                                                                ↓
点击提交      →  发送 POST 请求   →  RiskAgent 评估      →  Loading 状态
                                     ↓
                                                             成功提示
                                SearchAgent 检索预案
                                     ↓
                                返回风险评估 + 预案
```

## 常见问题排查

### 问题 1：前端无法连接后端

**错误信息：** `Network Error` 或 `ERR_CONNECTION_REFUSED`

**解决方法：**
```bash
# 检查后端是否运行
curl http://localhost:8000/health

# 正常响应：
{"status":"healthy","service":"ehs-ai-service","timestamp":"..."}
```

### 问题 2：Docker 容器无法启动

**错误信息：** `Address already in use`

**解决方法：**
```bash
# 停止占用端口的进程
docker-compose down
# 或手动停止
docker ps
docker stop <container_id>
```

### 问题 3：Python 依赖安装失败

**错误信息：** `SSL: CERTIFICATE_VERIFY_FAILED`

**解决方法：**
```bash
# 使用国内镜像
python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题 4：页面白屏

**解决方法：**
1. 检查控制台错误
2. 清除浏览器缓存
3. 重新安装依赖：`npm install`
4. 重启开发服务器
