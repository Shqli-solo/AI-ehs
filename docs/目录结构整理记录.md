## Monorepo 架构说明

### pnpm Workspace

```yaml
packages:
  - 'apps/*'
```

所有应用子包都在 `apps/` 目录下，pnpm 自动管理依赖。

### 应用子包

| 子包 | 技术栈 | 端口 | 描述 |
|------|--------|------|------|
| `apps/admin-console` | Next.js 14 + TypeScript | 3000 | 前端管理控制台 |
| `apps/ehs-ai` | FastAPI + LangGraph | 8000 | Python AI 服务 |
| `apps/ehs-business` | Spring Boot + COLA | 8080 | Java 业务服务 |

### 测试结构

- `tests/e2e/` - 端到端测试 (Playwright)
- `apps/ehs-ai/tests/` - Python 单元测试 (pytest)
- `apps/ehs-business/src/test/` - Java 单元测试 (JUnit)
